Active / Passive Voice (er / ero)
er
Placed before a noun to mark it as the SUBJECT (doer)
ero
Placed before a noun to mark it as the OBJECT (receiver)
💡 er and ero allow flexible word order (SVO, VSO, VOS, etc.). You can use both, one, or drop them entirely when context is clear.
5
Infinitive Prefix (ge-) & Event Construction

Add ge- before any verb to create its infinitive form.

Event Construction: [ge-verb + object] + sire

Plural & Intensification

Write a word twice, hyphen-separated, to make it plural. For adjectives/adverbs, this means an extreme degree.

Questions & Interrogatives

Counting System
Two-digit numbers: n + sif + m (e.g., 11 = ik + sif + ik = ikisik). Hundreds = sida, Thousands = toze, 10⁴ = idatoza, 10⁵ = sidatoza, 10⁶ = kaho.
